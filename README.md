# Yet!

Another instagram downloader build with Svelte.

### Installation

```
$ git clone https://github.com/husnulhamidiah/yet.git
$ cd yet
$ npm install
$ npm run dev
```

### Deployment

```
$ npm run build
$ git subtree push --prefix public origin gh-pages
```
